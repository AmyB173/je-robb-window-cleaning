# Testimonials

A bank of customer reviews for use across the J E Robb Window Cleaning website. All testimonials are from Google reviews, all 5-star.

## How to use this file

This file is the source of truth for testimonial copy on the site. When building or editing a page that uses customer quotes, pull from here rather than writing fresh copy or reusing the same quotes everywhere.

Two sections below:

1. **Snippets** — one and two-line quotes for hero carousels, trust bars, and short callouts.
2. **Full testimonials** — longer reviews for pull quotes and "Why People Keep Me On" style sections.

### Editing rules

- Reviews can be lightly edited for length (e.g. trimming a closing "Thanks Jack!") and to remove em dashes per CLAUDE.md voice rules. Don't change the reviewer's words or meaning.
- Don't combine quotes from different reviewers.
- Keep the reviewer's first name and last initial (or full name where it reads naturally) with each quote on the page.
- Don't reuse the same testimonial in more than one slot on the same page.
- Spread testimonial usage across different reviewers between pages where possible. If three area pages all use Anna Cairns' quote as the pull quote, that's a missed opportunity.

### Tags

Each entry has a short tag list. Use these to match testimonials to context:

- **first-clean** — reviewer was a new customer at the time
- **long-term** — reviewer has been a customer for a while, mentions ongoing service
- **detail** — mentions attention to detail, frames, sills, corners
- **communication** — mentions Jack messaging ahead, being easy to reach
- **conservatory** — mentions conservatory work
- **greenhouse** — mentions greenhouse work
- **value** — mentions price or value for money
- **reliable** — mentions reliability, turning up on time
- **friendly** — mentions Jack's personality, being polite or pleasant
- **recommend** — strong recommendation language
